# Trans-FARM
This is the readme file for the R scripts of the paper “Transfer Learning for High-dimensional Factor-augmented Sparse Linear Regression Model” by Bo Fu, and Dandan Jiang. Please make sure the packages needed for each R script are installed.

Install the following R packages: "glmtrans", "SIS", "mvtnorm", "glmnet", "doParallel", "doSNOW", "foreach", "clime". Please make sure the packages needed for each R script are installed.

Explanation of files:

1. The "Trans-FARM-Estimation.R" is the main R code to reproduce simulations of estimations in Section 5.1.
2. The "Trans-FARM-CI.R" is the main R code to reproduce simulations of constructing simultaneous confidence intervals in Section 5.2.
3. The "Trans-FARM-realdata.R" is the main R code to reproduce the real data analysis in Section 5.3.
